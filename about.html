운영 원칙과 검증 절차

calc-tools.kr은 한국의 노동법과 세법 계산을 정부 공식 자료 기준으로 정확하게 제공하는 정적 웹사이트입니다. 퇴직금, 주휴수당, 연차, 통상임금, 연봉 실수령액, 대출 상환, 복리 등 일상에서 자주 필요한 계산을 회원가입 없이, 광고 추적 없이, 개인정보 전송 없이 사용할 수 있습니다.

왜 이 사이트를 만들었나

한국에는 이미 수많은 계산기 사이트가 있습니다. 그러나 직접 사용해 보면 세 가지 공통된 불편이 있었습니다. 첫째, 적용 기준이 불투명합니다. 어떤 세율을 썼는지, 언제 갱신된 자료인지 알 수 없습니다. 둘째, 설명이 부족합니다. 결과 숫자만 보여주고 끝납니다. "왜 이 금액인지" 사용자가 검증할 방법이 없습니다. 셋째, 광고가 콘텐츠를 압도합니다. 계산 결과보다 광고가 더 많은 페이지가 흔합니다.

calc-tools.kr은 이 세 가지를 정반대로 풀어보려는 시도입니다. 모든 도구의 적용 기준과 출처를 페이지에 공개하고, 계산식과 법령을 함께 설명하며, 광고 영역을 페이지당 최소한으로 유지합니다. 실생활에서 필요한 계산을 직접 검증하고 사용하기 위해 만들어진 도구이기도 합니다.

네 가지 운영 원칙

① 정확성 — 정부 공식 자료만 사용

국세청, 고용노동부, 국민건강보험공단, 국민연금공단, 법제처 등 정부 공식 기관의 1차 자료만 출처로 사용합니다. 블로그·뉴스·2차 가공 자료는 인용하지 않습니다. 세율과 보험료율은 분기마다 자동 또는 수동으로 검토합니다.

② 개인정보 보호 — 클라이언트 사이드 계산

모든 계산은 사용자의 브라우저에서 자바스크립트로 직접 처리됩니다. 연봉·대출금·생년월일 등 어떤 입력값도 외부 서버로 전송되지 않습니다. 페이지를 닫으면 입력 내용은 즉시 사라집니다.

③ 투명성 — 공식과 출처 공개

각 도구 페이지에 적용한 계산 공식, 관련 법령 조문, 출처 URL을 함께 표시합니다. 사용자가 결과를 직접 검증할 수 있도록 모든 근거를 노출합니다.

④ 갱신 가능성 — 마지막 검토일 표시

각 페이지 하단에 "마지막 갱신일"과 "다음 검토 예정일"을 명시합니다. 법령 개정이나 요율 변경이 발생하면 신속히 반영하고, 사용자가 정보 신선도를 직접 확인할 수 있도록 합니다.

콘텐츠 검증 절차

새 도구를 만들거나 기존 도구를 갱신할 때 다음 4단계를 거칩니다. 사용자에게 보이지 않는 작업이지만, 결과의 신뢰성은 이 절차에서 나옵니다.

  1. 출처 확인 — 관련 법령(법제처) + 시행령 + 고시(국세청·고용노동부·건보공단)를 1차 자료에서 직접 확인합니다.
  2. 계산식 검증 — 법조문과 시행령에 명시된 공식을 그대로 코드로 옮기고, 정부 공식 모의 계산기(있는 경우)와 결과를 대조합니다.
  3. 실전 시나리오 적용 — 실제 사용 케이스 3개 이상을 직접 입력해 보고 결과가 합리적인지 점검합니다.
  4. 분기별 재검토 — 모든 핵심 도구는 분기마다 적용 세율·요율을 다시 검토하고, 변경이 있으면 페이지 상단에 갱신 이력을 추가합니다.

출처 데이터

본 사이트에서 사용하는 세율, 보험료율, 최저임금, 법정 기준 등 모든 핵심 수치는 다음 정부 공식 기관의 1차 자료를 출처로 합니다.

  • 국세청 (nts.go.kr) — 소득세 과세표준 구간 및 세율, 근로소득 간이세액표
  • 국민건강보험공단 (nhis.or.kr) — 건강보험료율, 장기요양보험료율
  • 국민연금공단 (nps.or.kr) — 국민연금 보험료율
  • 고용노동부 (moel.go.kr) — 최저임금, 고용보험료율, 근로기준법 적용 기준
  • 법제처 (law.go.kr) — 근로기준법, 퇴직급여보장법, 소득세법 등 법령 원문
