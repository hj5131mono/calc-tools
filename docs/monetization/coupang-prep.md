# 쿠팡 파트너스 준비 문서 (Phase 0)

> **주의**: 본 문서의 실제 가입·배너 노출은 **AdSense 4차 승인 확정 후** 집행. 심사 중 배너 삽입 절대 금지.

## 개요

- **채널 성격**: CPA/CPS (쿠팡 내 상품 구매 시 수수료 3%)
- **기대 수익**: 월 25~100만원 (트래픽·매칭도 따라)
- **AdSense와 관계**: 충돌 없음. 쿠팡 파트너스는 AdSense 거절 이력·정책과 무관. 동시 운영 가능.
- **가입 URL**: https://partners.coupang.com (사용자 수동 가입)
- **전제**: calc-tools.kr 도메인 소유권 증명 + 운영 이메일

## 가입 단계 (사용자 수동)

1. 파트너스 사이트에서 "회원가입" → 개인 사업자 또는 개인 선택
   - 익명성 유지를 위해 "개인" 선택 시 본명이 내부 기록만 남고 사이트에 노출되지는 않음
2. 프로필 등록 시 **표시명은 반드시 "calc-tools.kr 운영팀"** 또는 이메일 계정명 사용 — 본명·사진 업로드 금지
3. 사이트 정보 등록: https://calc-tools.kr / 카테고리 "라이프/금융"
4. 정산 계정 등록 (개인 계좌) — 내부 전용, 사이트 노출 없음
5. 승인 완료 후 대시보드에서 링크 생성 도구·상품 검색 기능 확인

## 2025.1 배너 규제 대응

- 2025년 1월부터 대형 배너형 광고(리플레이·와이드)는 "광고" 문구 노출 필수 + 콘텐츠 대비 과대 비율 제재
- calc-tools 대응:
  - **텍스트 링크 + 소형 카드형만 사용** (대형 배너 금지)
  - 각 링크 옆에 "쿠팡 파트너스 활동으로 수수료를 받을 수 있습니다" 문구 필수 (disclaimer.html + 각 페이지 푸터)
  - 카테고리 페이지 상단 고정 배너 절대 금지 (본문 하단·가이드 하단만)
