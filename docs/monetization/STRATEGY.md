2. **익명성 유지** — 모든 외부 제휴·계정·프로필은 "calc-tools.kr 운영팀" 명의 + ahnhj5131@gmail.com 단일 채널. GitHub 링크·실명·사진 노출 금지.
3. **사용자 본질 목적 align** — 삼쩜삼(환급액 증대), 쿠팡(실생활 편의), 핀다(대출 최적화) 같이 사용자 이익이 수익자와 align된 채널 우선. 클릭 유도·오염은 금지.
4. **법제처 1차 출처 흐름 훼손 금지** — 가이드·도구 콘텐츠 신뢰도를 외부 제휴 배너가 가리면 E-E-A-T 자해.
5. **피해야 할 옵션**: 네이버 애드포스트(외부 매체 미지원), 데이블·텐핑(콘텐츠 오염), AdThrive/Mediavine(PV 미달), 광고주 직접 접촉(익명성 훼손).
