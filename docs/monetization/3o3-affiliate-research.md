# 삼쩜삼 제휴 리서치 (Phase 0)

> **주의**: 실제 제휴 문의·링크 삽입은 **AdSense 4차 승인 확정 후**. 본 문서는 사전 조사·전략 수립 전용.

## 왜 삼쩜삼인가 (본질 align 분석)

- **사용자 본질 목적 1:1 매칭**: 사용자(안희준)가 밝힌 프로젝트 본질은 "근로소득 이외 미미하게나마 소득처 증대". 삼쩜삼은 사용자(방문자)의 **환급 누락분을 찾아주는 서비스** — calc-tools 방문 동기와 완벽히 일치.
- **수익 구조 = 사용자 이익과 동일 방향**: 방문자가 환급 더 받을수록 calc-tools도 수수료 증가. 클릭 유도·콘텐츠 오염 없이도 자연스러운 CTA 가능.
- **한국 특화도 100% 차별화 축과 align**: 삼쩜삼은 한국 국세청 홈택스 API 연동. 글로벌 경쟁사에는 없는 한국 전용 가치 채널.

## 서비스 개요

- **운영사**: 자비스앤빌런즈 (주)
- **서비스**: 홈택스 자동 연동 → 5년치 미환급 세금 자동 조회 → 환급 신청 대행
- **사용자 요금**: 환급액의 10~20% (성공 보수형)
- **가입자 규모**: 2025 기준 1,800만명 이상 (리서치 기준)
- **블로그·제휴 문의 경로**: https://blog.3o3.co.kr / 공식 홈페이지 하단 "제휴 문의"
- **AdSense 이력 영향**: 무관 (별도 CPA 네트워크)

## 예상 수익 구조

| 구분 | 수치 (리서치 추정) |
|---|---|
| CPA (가입 1건) | 3,000~8,000원 |
| 실제 환급 발생 시 추가 수수료 | 환급액의 1~3% (제휴 등급별) |
| 월 기대 수익 (4차 통과 후 year-end-tax 트래픽 기준) | 5~50만원 |
| 성수기 (1~3월 연말정산 시즌) | 2~5배 스파이크 |

## calc-tools 연계 방안

### 직접 매칭 (최우선)
- **year-end-tax 도구 결과 하단**: 계산 후 "환급액이 생각보다 적으신가요? 과거 5년치 미환급도 함께 조회해드립니다 → [환급 받으러 가기]"
- **연말정산 변경점 가이드 본문 하단**: "직접 계산이 어려우시면 홈택스 자동 연동 서비스를 이용하실 수 있습니다"
- **신용카드 공제 가이드**: 공제 한도 초과분을 정확히 집계해주는 도구로 자연스럽게 연결
- **부양가족 공제 가이드**: 과거 누락 부양가족 소급 공제 시나리오 (Grey Zone) → 삼쩜삼 자동 조회

### 본문 오염 방지 원칙
- 가이드 본문의 첫 1,000단어 이내 링크 삽입 금지
- 결론·요약 섹션 뒤에만 CTA 배치
- "광고·제휴 고지" 문구 필수 (disclaimer.html + 해당 페이지 푸터)
- 계산 결과 조작·과장 금지 — "더 받을 수 있다"는 조건부 안내만
