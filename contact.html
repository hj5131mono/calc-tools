문의하기

계산기 도구 모음을 이용해 주셔서 감사합니다. 계산 오류, 새로운 기능 제안, 사이트 이용 관련 문의 등 어떤 의견이든 환영합니다. 아래 안내에 따라 문의해 주시면 확인 후 최대한 빠르게 반영하겠습니다.

문의 방법

아래 GitHub 페이지에서 문의 내용을 남겨 주세요. GitHub 계정이 없어도 무료로 간편하게 가입하실 수 있습니다.

문의하기 (GitHub Issues)

문의 유형 안내

원활한 처리를 위해 문의 시 아래 유형 중 해당하는 항목을 함께 알려 주세요.

계산 오류 신고

특정 계산기의 결과가 부정확하거나 예상과 다른 경우. 입력값과 예상 결과를 함께 알려 주시면 빠르게 확인할 수 있습니다.

새 계산기 제안

추가를 원하시는 계산기나 도구가 있다면 알려 주세요. 많이 요청되는 기능부터 우선 개발합니다.

사이트 오류·개선

화면이 깨지거나, 버튼이 동작하지 않거나, 사용 중 불편한 점이 있으면 알려 주세요. 사용 기기와 브라우저 정보를 함께 남겨 주시면 도움이 됩니다.

광고·제휴 문의

광고 게재, 콘텐츠 제휴, 기타 비즈니스 관련 문의를 남겨 주세요.

자주 하는 문의

계산 결과가 다른 사이트와 다릅니다.
사이트마다 적용하는 세율, 공제 기준, 반올림 방식 등이 다를 수 있습니다. 계산기 도구 모음은 국세청·고용노동부 등 정부 공식 자료를 기준으로 계산하며, 각 페이지 하단에 적용 기준과 법적 근거를 안내하고 있습니다. 특정 수치가 의심되는 경우 해당 계산기 페이지명과 입력값을 알려 주시면 확인 후 답변드리겠습니다.
특정 계산기를 추가해 주세요.
GitHub Issues에 원하시는 계산기의 이름과 간단한 설명을 남겨 주세요. 요청이 많은 도구부터 우선적으로 개발합니다. 현재 제공 중인 도구 목록은 홈 페이지에서 확인하실 수 있습니다.
개인정보가 수집되나요?
계산기 도구 모음에서 입력하시는 모든 데이터(연봉, 대출 금액, 생년월일 등)는 사용자의 브라우저에서만 처리되며, 외부 서버로 전송되지 않습니다. 사이트 방문 분석을 위해 Google Analytics를 사용하고 있으며, 자세한 내용은 개인정보처리방침을 참고해 주세요.
모바일에서 화면이 깨집니다.
사용 중인 기기명, 브라우저(Chrome, Safari 등), 문제가 발생하는 페이지를 함께 알려 주시면 빠르게 수정할 수 있습니다. 최신 버전의 Chrome, Safari, Samsung Internet 브라우저를 권장합니다.

문의 내용은 확인 후 가능한 빠르게 반영하고 있습니다. 보내 주신 의견은 서비스 개선에 큰 도움이 됩니다. 감사합니다.
